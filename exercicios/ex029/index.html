Peso, Estilo e Shorthand font

Peso tem relação com a família tipográfica de uma fonte.
Fonte usada para mexer com peso: "Work Sans"
Assim ficando: 


    font-family: "Work Sans"

Obrigatório para peso de fonte: 
font-weight:...

Diferença entre Weight, Width e height

  • Weight = Peso
  • Width = Largura
  • Height = Altura

Pesos aceitáveis por fontes

Lighter, normal, bold, bolder;
OBS: Nem todas as fontes aceitam todos esses pesos.
Também é possível utilizar pesos numéricos.
Ex: 100-900

Font Style e Combinações Utilizadas

Para font style: Normal ou Italic
Para texto sublinhado: 

text-decoration: undeline

Combinacão bastante utilizada

            
                h1 { 

                   font-family: "Work-Sans", saans-serif; 
 
                   font-Weight: bolder; 
 
                   fonte-size: 3em; 
 
                   font-style: italic; 

                }

Oque é ShortHand e como Utilizá-lo?

O Shorthand permite que você escreva aquela combinaão bastante utilizada no h1
seguindo uma única linha, assim ficando a sequência:
/* Shorthand font
font-style -> font-Weigt -> font-size -> font-family */
font: italic bolder 3em "Work Sans", sans-serif
