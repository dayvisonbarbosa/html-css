Tamanho de Fonte e Suas Medidas

Medias Absolutas

Cm, mm, in, px, pt, pc.

  • Cm = Centímetro
  • Mm = Milímetro
  • In = Polegada
  • Px = Pixel
  • Pt = Ponto
  • Pc = Paica

Medidas Relativas

Em, ex, reon, vw, %.

  • Em -> Medida relativa ao tamanho atual da fonte
  • Ex -> Relativa à altura X de uma fonte
  • Reon -> Relativa à fonte que está configurada no < body >
  • Vw -> Largura da ViewPort
  • % -> Porcentagem

Oque é ViewPort?

ViewPort nada mais é que o tamanho da sua tela

Mais de fontes

Recomendações para mexer com tamanho de fonte: px, em;
Tamanho normal de fonte: 16px;
Obs: 16px = 1em

Código para Font-family e Font-size

            
                < style >
                        body {
                            font-family:
                            font-size: 16px
                        }
                < /style >
