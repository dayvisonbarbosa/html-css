Links Internos

Links Internos são como por exemplo outras páginas do meu próprio site.

Etapas para criar outras páginas
para meu site

- Crie o arquivo pag002.html em seu exercício;
- Após, use CTRL + Espaço para colocar o arquivo da página 2 dentro do meu href=;
- Use rel="next" para indicar que você está indo para a próxima página, e use
rel="prev" para indicar que você está
retornando para a página anterior.

rel="nofollow" - Oque Significa?

O atributo rel="nofollow" está dizendo aos mecanismos de busca que não devem seguir esse link,ou seja:
- Não transferir a autoridade do meu site para o site de destino;
- Não usar esse link para influenciar o ranqueamento da página de destino.

Criando uma página dentro de uma pasta

E se eu colocar uma página (Exemplo: pag003.html)dentro de uma pasta ? (Exemplo: notícias)
Ao colocar um arquivo de página dentro de uma pasta, se você tentar apenas colocar o nome do seu arquivo dentro do href=""
ele não será encontrado pois agora ele está em outro diretório, por isso
use CTRL + Espaço para localizar o diretório da sua pasta.

Como voltar para a primeira ou para a segunda página, se eu estiver em uma página dentro de
uma pasta e minhas outras páginas estiverem dentro
da primeira página?

Para entender melhor a sua dúvida, aqui vai um exemplo:
Você criou uma primeira pasta chamada ex013, onde lá está o arquivo index.html e outra(s) página(s),
E também junto com esses dois arquivos, você criou uma pasta já dentro da pasta 013 para colocar
outras páginas (Exemplo: 013/index.html e pag002.html e notícias/pag003), então para conseguir retornar para a primeira pasta (ex013) você deve:
- Usar ../ para voltar uma pasta, assim ficando:
href="../index.html" -> Retorna para a primeira página.
