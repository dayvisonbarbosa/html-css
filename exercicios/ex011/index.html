Listas Mistas e Definição

Listas Mistas

Listas mistas são aquelas que misturamos listas ordenadas com não ordenadas.

Exemplo de Listas Mistas Abaixo:

Meus jogos favoritos

  1. Nes
    • Mario Bross 3
    • Mario: Lost Levels
  2. Ninja Gaiden
  3. SNES
    • Mario
    • Donkey Kong
  4. PlayStation
    • Final Fantasy
    • Castlevania

Colocar uma lista dentro da outra é bastante usado para a criação de menus.

Listas de definição

São usadas para dizer oque significa um termo ou alguma coisa, exemplo:
HTML
Linguagem de marcação de hypertexto para a criação de site.
CSS
São usadas para a criação do desing de um site.
JavaScript
Linguagem de programação usada para criar interatividade em um site.
