Listas Ol e Ul

Listas Ordenadas

As listas ordenadas são aquelas que usamos para rotina, ou seja, uma série de etapas organizadas de maneira cronológica.

Exemplo de Lista Ordenada

-Rotina da semana

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Se arrumar para o trabalho
  5. Ir trabalhar

O parâmetro type

A tag Ol possui um parâmetro type, onde configura o tipo de marcador da lista atual.
As opções de valores para os parâmetros são:
1- Valor padrão, cria listas numeradas. Ex: 1, 2, 3, 4...
A- Cria listas alfabéticas maiúsculas. Ex: A, B, C, D...
a- Cria listas alfabéticas minúsculas. Ex: a, b, c, d...
I- Cria listas com números romanos maiúsculos. Ex: I, II, III, IV...
i- Cria listas com números romanos minúsculos. Ex: i, ii, iii, iv...
Exemplo: Ol type= "I" start = "5" -> Vai gerar itens numerados com números romanos maiúsculos, começando em 5 (V)

Listas Não Ordenadas

As listas não ordenadas são aquelas que não seguem uma ordem cronológica como uma rotina, podendo ser uma lista de compras ou algo que pode ser feito sem seguir uma ordem obrigatória.

Exemplo de Lista Não Ordenada

Lista-De-Compras

  • Pão
  • Leite
  • Café
  • Requeijão
  • Manteiga
  • Queijo

Marcadores

O marcador padrão é uma bolinha preta totalmente preenchida, mas existe a opção de configurar a propriedade type da tagt ul com os seguintes valores:
1- disc
2- circle
3- square
