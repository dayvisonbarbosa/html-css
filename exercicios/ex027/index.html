Anatomia do Tipo

Oque são Fontes?

São um conjunto de glifos (A, B, C..Z).

Oque são glifos?

São as letras do alfabeto em uma fonte.
Às vezes uma fonte tem diferentes maneiras de representar um glifo, é oque chamamos de
família tipográfica (variação)

Catagoria de fontes

  • Serifada -> Fonte que tem serifa.
  • Sans-Serif -> Fontes que não tem serifa.
  • Monoespaçados -> Fontes em que cada letra (glifo) tem o mesmo espaço de comprimento.
  • HandWriting -> Fontes que simulam a letra humana.
  • Display -> Fontes que não apresentam nenhuma das características acima.
